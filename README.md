# Danmaku Tube

A browser-based bullet hell game synchronized with video timelines.

## Overview

Danmaku Tube is a browser game that generates bullet patterns synchronized with the timeline of a video or music track.

Players dodge bullets while enjoying rhythm and visual patterns that evolve in sync with the timeline.

## Features

- Timeline-based bullet generation
- Multiple bullet pattern systems
- Mouse and keyboard controls
- Loop system for continuous gameplay
- Song and stage selection UI
- Highly customizable attack patterns

## Technologies Used

- JavaScript
- Phaser
- Vite
- YouTube IFrame Player API

## Current Implementations

- Homing bullets
- Wave bullets
- Circular bullet patterns
- Spiral bullet patterns
- Vertical and horizontal lasers
- Timeline synchronization system
- Full-screen mode
- Stage selection screen
